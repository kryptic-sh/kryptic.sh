kryptic.sh

Native software tools for people who never left the terminal. Vim-shaped, Rust-built. No Electron, no JVM.

projects

  • sqeel Vim-native SQL client. MySQL · Postgres · SQLite. TUI + GUI from one codebase.
  • buffr Vim-inspired browser. Native, GPU-accelerated. Rust + CEF.
  • hjkl Vim engine, rope buffer, and modal editor primitives. Rust, no_std + alloc.

principles

  • Native Rust. No JVM, no Electron, no webview wrappers.
  • Vim-shaped. Modal editing as a first-class citizen.
  • Mouse and keyboard, both, always.
  • No telemetry. No analytics. Just tools.
